# Steering Sync Output Contract

## Required Sections

```markdown
## Mode

Sync

## Changes

- [file name]: [summary of changes]
- ...

## Code Drift

- [description of drift]
- ...

## Recommendations

- [suggestions such as additional custom steering files to create]
- ...
```

## Quality Criteria

| Criterion | Verdict |
|-----------|---------|
| Detected drift is described concretely | Required |
| User sections are preserved | Required |
| Updates are additive | Required |
| No security information is included | Required |
| updated_at timestamp is present | Recommended |
