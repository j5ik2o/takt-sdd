# Custom Steering Output Contract

## Required Sections

```markdown
## Created File

- .kiro/steering/{name}.md

## Base

- Template: [template name or "generated from scratch"]
- Analysis Target: [directories or file patterns analyzed]
- Extracted Patterns: [key patterns detected]

## Content Overview

- [Section 1 overview]
- [Section 2 overview]
- ...

## Next Steps

[Prompt review and customization]
```

## Quality Criteria

| Criterion | Verdict |
|-----------|---------|
| One file covers one topic | Required |
| Patterns are documented (not mere lists) | Required |
| No security information is included | Required |
| Stays within 100-200 lines | Recommended |
| Does not overlap with core steering (product/tech/structure) | Required |
| Includes concrete code examples | Recommended |
| Follows template structure (when a template exists) | Required |
