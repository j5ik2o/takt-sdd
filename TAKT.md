# TAKT Development Policy

This file records TAKT-side conventions for this repository.

- `.agents/skills/kiro-*` are upstream Kiro skill assets. Do not directly modify or fork them for project-local behavior.
- TAKT workflows and facets may cover upstream skill ambiguity, outdated wording, or missing guardrails through higher-priority runtime prompts.
- `.kiro/settings/**` is project-local configuration and may be changed when a spec requires it.
